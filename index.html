Fraud transaction detection

Role: Data analyst

Description: This is a hybrid deep learning model using both supervised and unsupervised learning branches for detecting the fraudulent transactions from any given set of transactions. I tested it on a cryptocurrency transaction dataset.For the unsupervised learning part I used self organizing map to detect the clusters of transactions. By setting a threshold value the outlying clusters detected were the ones suspected to be fraudulent. Then using this data a dependent variable was then created which could be used to train a simple artificial neural network the supervised learning branch. This predicted a list of customers involved in the fraudulent transactions and their associated probability of involvement in frauds. It was able to produce results of accuracy upto 95 percent.
